emacs总结

教程来源

快捷键显示方式用法：
  1. “C-x”表示按住control的同是按住x键
  2. “M-x”表示按住alt的同时按住x键（Mac上没有用）/按一下esc再按一下x
快捷键总结：
  • C-g: quit
  • C-x C-c: exit
  • C-x C-f: opne a file
  • C-x C-s: save file
  • C-f/C-b: move forword/back a character
  • M-f/M-b: move forword/back a word
  • C-d/M-d: delete a character/word
  • C-_: undo
  • C-p/C-n: Move up/Down to the previous line
  • C-a/C-e: Move to the beginning/end line
